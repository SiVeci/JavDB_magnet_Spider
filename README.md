# 🧲 JavDB Magnet Scraper (智能磁力优选爬虫)

这是一个基于 Python 编写的 JavDB 自动化爬虫工具。它不仅能批量抓取指定演员或标签的影片目录，还能深入详情页，根据自定义的优先级规则（如：中文字幕、无马赛克、文件大小等），在众多磁力链接中**自动为你挑选并保存质量最高的一个**。

## ✨ 核心特性 (Features)

* 🛡️ **无视 Cloudflare 盾**: 底层弃用传统的 `requests`，改用 `curl_cffi` 完美模拟真实 Edge/Chrome 浏览器的 TLS 指纹，稳定绕过 403 Forbidden 拦截。
* 🧠 **智能优选算法**: 面对同一部影片的几十个磁力版本，程序会根据严格的优先级自动评估并提取最佳资源：
  * `无码+中字 > 中字 > 无码 > 高清 > 其他`
  * 在同等标签下，自动对比发布日期（取最新）和文件大小（取最大）。
* ⚙️ **稳健的两步走架构**:
  * **Step 1**: 遍历目录页，提取所有影片的番号和 URL，生成初始清单。
  * **Step 2**: 读取清单，深入详情页抓取最优磁力。即使网络意外中断，第二步也能基于已有的 CSV 继续执行，无需从头再来。
* 📊 **CSV 数据持久化**: 抓取结果（包含影片番号、原标题、最佳资源文件名、磁力链接、文件大小等）会自动规范化保存为 `.csv` 格式。

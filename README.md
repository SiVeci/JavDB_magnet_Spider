
# 🧲 JavDB Magnet Scraper

这是一个基于 Python 编写的 JavDB 自动化爬虫工具。它不仅能批量抓取指定演员或标签的影片目录，还能深入详情页，根据自定义的优先级规则（如：中文字幕、无马赛克、文件大小等），在众多磁力链接中**自动为你挑选并保存质量最高的一个**。

## ✨ 核心特性 (Features)

* 🛡️ **无视 Cloudflare 盾**: 底层弃用传统的 `requests`，改用 `curl_cffi` 完美模拟真实 Edge/Chrome 浏览器的 TLS 指纹，稳定绕过 403 Forbidden 拦截。
* 🧠 **智能优选算法**: 面对同一部影片的几十个磁力版本，程序会根据严格的优先级自动评估并提取最佳资源：
  * `无码+中字 > 中字 > 无码 > 高清 > 其他`
  * 在同等标签下，自动对比发布日期（取最新）和文件大小（取最大）。
* ⚙️ **稳健的两步走架构**:
  * **第一阶段**: 遍历目录页，提取所有影片的番号和 URL，生成初始清单。
  * **第二阶段**: 读取清单，深入详情页抓取最优磁力。即使网络意外中断，也能基于已有的 CSV 继续执行。
* 📊 **CSV 数据持久化**: 抓取结果自动规范化保存为 `.csv` 格式，完美兼容 Excel。

## 🚀 快速开始 (Quick Start)

### 1. 环境准备
请确保你的电脑已安装 Python 3.8+，然后在终端运行以下命令安装必需的依赖库：
```bash
pip install curl_cffi beautifulsoup4
```

### 2. 爬虫参数配置 (必读 ⚠️)
在执行代码前，必须正确配置 `START_URL`、`Cookie` 和 `User-Agent`，否则会被网站拦截。请按照以下步骤操作：

#### 步骤 A：获取目标链接 (START_URL)
1. 在浏览器中登录 JavDB。
2. 搜索并点击进入你想要爬取的**某位老师的主页**。
3. 在页面上方点击筛选你想要的 **Tag**（例如选择“单体作品”、“高清”等）。
4. 待页面刷新后，直接**复制浏览器地址栏中的完整链接**。
5. 将链接粘贴到代码的 `START_URL = '...'` 处。

#### 步骤 B：获取 Cookie 和 UA (以 Edge 浏览器为例)
**注意：Cookie 和 User-Agent 必须配套使用！**
1. 停留在你刚才打好 Tag 的网页上，按下键盘的 `F12` 键打开“开发者工具”。
2. 切换到顶部的 **网络 (Network)** 标签页。
3. 按 `F5` 刷新当前网页。
4. 在左侧抓取到的网络请求列表中，点击最上面第一个请求（通常就是当前网页的地址）。
5. 在右侧弹出的详情面板中，向下滚动找到 **请求标头 (Request Headers)** 区域：
   * **获取 Cookie**：找到 `cookie:` 字段，复制冒号后面的**所有长串字符**，填入代码。
   * **获取 UA**：找到 `user-agent:` 字段，复制后面的字符串，填入代码。

#### 步骤 C：配置本地代理 (可选)
如果你需要通过代理访问该网站，请在代码的 `PROXIES` 字典中填入你本地代理软件的端口（如 `http://127.0.0.1:7890`）。

---

### 3. 执行脚本
只有在完成以上所有参数的替换后，才可以开始运行脚本：

**第一步：抓取目录清单**
```bash
python step1_list.py
```
> 运行结束后，会在当前目录下生成 `step1_list.csv`。

**第二步：提取最优磁力链接**
```bash
python step2_get_magnets.py
```
> 程序会自动读取清单并开始筛选，最终完美的数据将保存在 `final_magnets.csv` 中。
